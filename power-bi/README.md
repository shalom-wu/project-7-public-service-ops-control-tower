# Power BI Package

## ⚠️ Why there is no .pbix file in this folder

**Power BI Desktop is not installed in the environment this project was built
in** (verified: no `PBIDesktop.exe`, no Store app), and a `.pbix` cannot be
authored programmatically — Microsoft provides no supported way to build
report visuals outside Power BI Desktop. Rather than ship a fake or empty
placeholder file, this folder contains everything needed to build the
dashboard in **30–45 minutes** on any machine with the free Power BI Desktop:

| File | Purpose |
|---|---|
| [`manual_build_instructions.md`](manual_build_instructions.md) | Exact click-by-click steps to recreate the 4-page dashboard |
| [`dashboard_brief.md`](dashboard_brief.md) | What the dashboard is for, page by page, and the headline insight |
| [`data_model.md`](data_model.md) | Star-schema model, relationships, and diagram |
| [`dax_measures.md`](dax_measures.md) | Every DAX measure, copy-paste ready |
| [`refresh_instructions.md`](refresh_instructions.md) | How to refresh the data end to end |
| [`screenshots/`](screenshots) | **Design mockups** of all 4 pages, rendered from the real data (clearly labeled — they are page-layout mockups, not Power BI screenshots) |

The **Power BI-ready data** lives in [`../data/powerbi/`](../data/powerbi):
one request-level fact table (Parquet, 1.07M rows — Power BI Desktop reads
Parquet natively) plus pre-aggregated KPI tables and dimensions (CSV), all
regenerated by `python scripts/run_sql.py`.
